## 开始开发

```bash
$ git clone https://github.com/it-blacklist/it-blacklist.github.io.git
```
安装依赖

```bash
$ yarn install
```

开始构建

```bash
$ yarn run serve
```

## 发布

```bash
$ yarn run build
```
